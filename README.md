# Be aMazed

#### What is this?
Be amazed is a simple and fun endless maze-runner game, that entails chasing a point in an ever-changing maze, until your time runs out

#### How do I play?
Simply use your arrow keys to navigate the player(the pink dot) around the maze, to reach the Origin(blue square)! Once you've reached it, the maze will MAGICALLY SHIFT, leaving you with another Origin.

## How do I build this?

It's as easy as copying the files from this git into a C compatible IDE (such as VSCode or CLion, ensuring that CMake libraries are enabled).
If you are on linux, I recommend you install the necessary dependencies, by copying this command into your terminal(if you're on an Ubuntu based dist
sudo apt install libx11-dev libxrandr-dev libxinerama-dev libxcursor-dev libxi-dev libxext-dev libgl1-mesa-dev libglu1-mesa-dev
